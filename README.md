# bank_management_system# 🏦 Bank Management System (C++)


This is a simple Bank Management System built using C++.  
It allows users to perform basic banking operations through a menu-driven program.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)

## 💡 What I Learned
- Basics of OOP in C++
- Handling user input and logic building
- Designing menu-driven program
